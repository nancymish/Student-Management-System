Student Management System 
A full-stack MERN (MongoDB, Express, React, Node.js) web application for managing students with role-based dashboards for Admin and Student. This project allows Admin to manage student data (Create, Read, Update, Delete) and Students to view and edit their profiles.

Live Demo

Frontend (React App):(https://student-management-system-topaz-chi.vercel.app/)

Backend (Node.js API): https://student-management-system-wf67.onrender.com

Tech Stack

Frontend: React, Tailwind CSS, React Router

Backend: Node.js, Express.js

Database: MongoDB (Atlas or local)

Authentication: JWT (JSON Web Token)

Deployment: Vercel (frontend), Render (backend)
## Run Locally
```bash
git clone "https://github.com/nancymish/Student-Management-System.git"
cd Student Management System

# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm run dev
